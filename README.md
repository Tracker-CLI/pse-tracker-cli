# Installation

You need to install [Node.js](https://nodejs.org/en/download/) first, then install the tool globally using this command:

```bash
npm install -g pse-tracker-cli
```
<p align="center"><img src="/.doc/install.gif?raw=true"/></p>


# Usage

```sh-session
$ npm install -g pse-tracker-cli

$ pse [SYMBOL]
running command...

$ pse --help [COMMAND]

USAGE
  $ pse [SYMBOL]
  $ pse --stock=symbol
...
```
